# Installation

## Prerequisites

- iOS 13.0 or macOS 10.15
- Swift 5.5

## Swift Package Manager

In your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/immutable/imx-core-sdk-swift.git", from: "0.1.0")
]
```
