## Immutable Core SDKs

The Immutable Core SDKs provide convenient access to the Immutable APIs and Ethereum Contract methods for applications that want to integrate with Immutable. The Core SDK is presently available in Typescript and will be available in many languages in the near future.

## Immutable Wallets SDKs

The Immutable Wallets SDKs provide a straightforward way to connect with a range of L1 and L2 wallets, in order to get access to the needed signers for the Core SDKs functionalities.
